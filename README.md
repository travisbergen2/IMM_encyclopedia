# The IMM Encyclopedia

Britannica-register reference work for the Information Manifold Model research
programme. Every article carries a status chip drawn from Paper 0B's Claim
Ledger vocabulary (STABLE / CONJECTURE / OPEN / FALSIFIED / NEGATIVE RESULT),
and a standing article catalogues the programme's retired and falsified claims.

**Live:** https://travisbergen2.github.io/IMM_encyclopedia/

- `index.html` — the encyclopedia (self-contained, no build step)
- Entries are grounded in the published corpus at earned epistemic grade;
  where a claim's status is contested, the Claim Ledger wins.
